# blueberry

# Setup
```
pip install git+https://github.com/lumiralabs/blueberry.git@main
```

Available Commands:

1. 
```
blueberry init <name>
```

creates a folder structure with:
```
<name>
    - frontend
        - pages/...
        - api/...
        - package.json
    - supabase
        - migration files
```
